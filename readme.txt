== Description ==

With the **SimpleGal** plugin you can simply create galleries.  
Just upload your images, add the **SimpleGal** Shortcode to your posts or pages and the gallery will appear.
For an optimal gallery view you should install a lightbox plugin.  
  
Mit dem **SimpleGal** Plugin kannst du ganz einfach Galerien erstellen.  
Einfach deine Bilder hochladen, den **SimpleGal** Shortcode deinem Artikel oder deiner Seite hinzuf&uuml;gen und schon ist deine Galerie fertig.
F&uuml;r eine optimale Galerie-Darstellung solltest du ein Lightbox-Plugin installieren.   
  
****

**Available languages:** English and German

== Installation ==

This section describes how to install the plugin and get it working.  


1.  Upload \`simplegal\` to the \`/wp-content/plugins/\` directory

2.  Activate the plugin through the Plugins menu in WordPress

3.  Go to **SimpleGal**-Option page and configure the plugin

  
For more information about how to use see also **SimpleGal**-Option page.
